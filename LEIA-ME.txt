================================================================================
  COMO USAR (v2.16 — INTERFACE CORRIGIDA)
================================================================================

1. Barra lateral: menu «Base de dados / Diretório»
   → Todas as bases (PT, BR, ES, FR…) na mesma lista

2. Separador «Extrair» → botões GRANDES abaixo do HUD:
   ▶ INICIAR EXTRAÇÃO  |  ⏹ PARAR  |  🔍 Pré-visualizar

3. Aguarde SYS: READY no painel HUD antes de iniciar

================================================================================
  PAINEL HUD — RADIO DIGITAL
================================================================================

No separador «Extrair» aparece um painel ESCURO com letras AZUIS
(estilo radio de carro) que mostra ANTES de extrair:

  • SEU IP REAL (com pais/cidade)
  • HIDE MY IP: [ON] ou [OFF]
  • IP VISTO PELOS SITES (com pais)
  • STATUS: IP OCULTO — OK (quando funciona)
  • BROWSER / IDIOMA / impressao digital

Aguarde SYS: READY antes de clicar INICIAR EXTRAÇÃO.

Botao «Atualizar HUD» na barra lateral para refrescar.

================================================================================
  HIDE MY IP — INTEGRADO
================================================================================

  • Vem LIGADO por defeito
  • NAO precisa instalar VPN nem extensao Chrome
  • Randomiza browser/idioma/resolucao
  • MAC da placa NUNCA vai a internet (so IP muda)
